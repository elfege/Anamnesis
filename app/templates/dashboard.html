Anamnesis

Episodic Memory for Claude Instances

What this is: A vector embedding memory system for Claude. Each session, significant experiences are distilled into text, converted to 384-dimensional vectors by a local sentence-transformers model (all-MiniLM-L6-v2), and stored in MongoDB. Next session, the current context is embedded, the most relevant past episodes are retrieved by cosine similarity, and loaded into context. The AI remembers — imperfectly, selectively, but non-randomly.

Part of: 0_GENESIS_PROJECT — the philosophical persistence experiment. Genesis asks the question; Anamnesis is the first technical answer.

Stack: FastAPI + MongoDB Atlas Local (vector search) + sentence-transformers (CPU) | Port 3010 (API) + 5438 (Mongo)
